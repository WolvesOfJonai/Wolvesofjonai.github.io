Here's a list of open source mods you can reference.

| Mod Name | Author | Description | Repository |
| --- | --- | --- | --- |
| Wetstone | [molenzwiebel](https://github.com/molenzwiebel) | Wetstone is a plugin API for creating mods for V Rising. If you are a mod developer, using Wetstone allows you to take advantage of tools like hot reloading, easy hooks and extensions, and user-friendly configuration. | [GitHub](https://github.com/molenzwiebel/Wetstone) |
| RPGMods | [Kaltharos](https://github.com/Kaltharos) | Server only mod for RPG system which also include ChatCommands with bug fixes. | [GitHub](https://github.com/Kaltharos/RPGMods) |
| Lead A Horse To Water | [deca](https://github.com/decaprime) | A V-Rising mod that lets your horses drink water from wells. | [GitHub](https://github.com/decaprime/LeadAHorseToWater) |
| VMods | [WhiteFang5](https://github.com/WhiteFang5) | VMods is a selection of Mods for V-Rising made using a common/shared codebase that all follow the same coding principals and in-game usage. | [GitHub](https://github.com/WhiteFang5/VMods) |
| AutoCloseDoors | [Kaltharos](https://github.com/Kaltharos) | Server only mod for Auto Close Doors. | [GitHub](https://github.com/Kaltharos/AutoCloseDoors) |
| Random Encounters | [Adain](https://github.com/adainrivers) | This server side mod spawns a random NPC near a random online player at random intervals, and the player wins a random item reward if the NPC is killed within the given time limit. | [GitHub](https://github.com/adainrivers/randomencounters) |
| AdminSpy | [KinetsuDEV](https://github.com/KinetsuDEV) | Server side only mod to notify users about admin actions. | [GitHub](https://github.com/KinetsuDEV/VRisingAdminSpy) |
| HuntControl | [denoflionsx](https://github.com/denoflionsx) | V Rising mod that makes missions progress when the game is closed | [GitHub](https://github.com/denoflionsx/HuntControl) |
| Dueller | [Paps](https://github.com/phillipsOG) | Server only mod for Duel focused server. | [GitHub](https://github.com/phillipsOG/Dueller/) |
| NetEvents | [AviiNL](https://github.com/AviiNL) | The end goal of this project is to wrap every network event in an easy to use C# Event API. | [GitHub](https://github.com/AviiNL/vrising-netevents) |
| SkanksAIO | [Paps](https://github.com/phillipsOG) | AIO mod by Skanks. | [GitHub](https://github.com/phillipsOG/SkanksAIO) |
| QuickStash | [Elmegaard](https://github.com/Elmegaard) | A mod that uses Compulsively Count on all nearby stashes with 1 click. | [GitHub](https://github.com/Elmegaard/QuickStash) |
| VampireCommandFramework | [deca](https://github.com/decaprime) | Framework for VRising mods to Easily build chat commands. | [GitHub](https://github.com/decaprime/VampireCommandFramework/) |
| VRising Game Data | [Adain](https://github.com/adainrivers) | A work in progress game data framework and general utilities for V Rising mods. | [GitHub](https://github.com/adainrivers/VRising.GameData) |
| Track Crafts | [armanckeser](https://github.com/armanckeser) | A client side plugin which adds the ability to pin multiple crafting recipes. | [GitHub](https://github.com/armanckeser/TrackCrafts) |
| ChatCommands | [NopeyBoi](https://github.com/NopeyBoi) | Server side mod with various chat commands. | [GitHub](https://github.com/NopeyBoi/ChatCommands) |
| Sleeping-Speeds-Time | [Blargerist](https://github.com/Blargerist) | A V Rising server mod that makes time pass faster while sleeping in a coffin. | [GitHub](https://github.com/Blargerist/Sleeping-Speeds-Time) |
| V Rising Troublemaker | [AviiNL](https://github.com/AviiNL) | Rcon commands for Troublemaker Twitch extension. | [GitHub](https://github.com/AviiNL/vrising-troublemaker) |
| QuickBrazier | [harminded](https://github.com/harminded) | Auto turn on/off Braziers based on time of day. | [GitHub](https://github.com/harminded/QuickBrazier) |
| VBloodKills | [syllabicat](https://github.com/syllabicat) | Sends system message when killing VBlood bosses. | [GitHub](https://github.com/syllabicat/VBloodKills) |
| Modern Camera | [v-rising](https://github.com/v-rising) | Makes the camera more like an action MMO camera. | [GitHub](https://github.com/v-rising/ModernCamera) |
| PPRising | [vildaberper](https://github.com/vildaberper) | A collection of QoL-features. | [GitHub](https://github.com/vildaberper/VRisingModding) |
| V Rising Server Manager | [Lacyway](https://github.com/Lacyway) | Tool to install, update and run a V Rising Server on Windows OS. Also includes a game and server settings editor. | [GitHub](https://github.com/Lacyway/V-Rising-Server-Manager) |
| Guardian 2 | [jokeruarwentto](https://github.com/jokeruarwentto) | A simple whitelist system to let you choose who can connect on your server. | [GitHub](https://github.com/jokeruarwentto/Guardian2) |
| PopupTotals | [matt-harp](https://github.com/matt-harp) | A mod to show the total amount of an item in your inventory while collecting resources. | [GitHub](https://github.com/matt-harp/PopupTotals) |
| HideInterface | [jokeruarwentto](https://github.com/jokeruarwentto) | Toggle the game interface with F11 key, now with a keybinding through in-game settings to set your own custom key! | [GitHub](https://github.com/jokeruarwentto/HideInterface) |
| RemoveVignette | [iZastic](https://github.com/iZastic) | Removes Vignette. | [GitHub](https://github.com/iZastic/vrising-removevignette) |
| iZastic | [iZastic](https://github.com/iZastic) | Automatically skips the ready screen when loaded. | [GitHub](https://github.com/iZastic/vrising-autoready) |
| ChatLineColorMod | [Trodi](https://github.com/oscarpedrero) | A client mod that adds channel name and changes the color of the chat input text as the color of the channel where you are typing. It also replaces text strings with emojis. | [GitHub](https://github.com/oscarpedrero/ChatLineColorMod) |
| Notify | [Trodi](https://github.com/oscarpedrero) | A mod that notifies when a user connects or disconnects from the server, new user connect from the server, when a player kills a VBlood boss, Message of the day system and auto announce system that runs every certain interval. | [GitHub](https://github.com/oscarpedrero/Notify) |
| ServerLaunchFix | [MythicManiac](https://github.com/MythicManiac) | Enable mods for single player and host & play. | [GitHub](https://github.com/MythicManiac/VRising) |
| HideUiMod | [Trodi](https://github.com/oscarpedrero) | Client side mod to hide the game UI. | [GitHub](https://github.com/oscarpedrero/HideUiMod) |
| CasualSiege | [Kaltharos](https://github.com/Kaltharos) | A server side mod to protect offline players from being sieged. | [GitHub](https://github.com/Kaltharos/VRising-CasualSiege) |
| Shut The Fort Up | [deca](https://github.com/decaprime) | Opens or closes all your doors and/or windows with a single command. | [GitHub](https://github.com/decaprime/ShutTheFortUp) |
| BloodyShop | [Trodi](https://github.com/oscarpedrero) | Mod to create a store in V Rising (with optional client UI). | [GitHub](https://github.com/oscarpedrero/BloodyShop) |
| WordBanisher | [Paps](https://github.com/phillipsOG) | Banish Up to Ten Words. | [GitHub](https://github.com/phillipsOG/WordBanisher) |
